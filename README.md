# OB/GYN Master

An interactive, evidence-based learning platform for obstetrics and gynecology education.

## Features

- **7 Comprehensive Modules**: Prenatal Care, Hypertensive Disorders, Labor & Delivery, Fetal Monitoring, Postpartum Complications, Gynecologic Emergencies, and Infections
- **Interactive Assessments**: Test your knowledge with MCQs and multi-select questions with detailed rationales
- **Mobile-First Design**: Optimized for learning on any device
- **Dark Mode**: Easy on the eyes during long study sessions
- **Offline Support**: Access content without an internet connection
- **Progress Tracking**: Monitor your learning journey

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Zustand (state management)
- React Router (navigation)
- IndexedDB (offline storage)
- PWA (Progressive Web App)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
